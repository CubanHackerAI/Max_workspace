## CrossAuto - Windows GUI Automation Bridge

Created a Node.js server to control Windows GUI from WSL. Location: `D:\code\crossauto`

### Key Endpoints
- `POST /cursor/open` - Launch Cursor IDE on a folder
- `POST /type` - Type text
- `POST /key` - Press key combos (e.g., `["ctrl","l"]`, `["enter"]`)
- `POST /cursor/screenshot` - Focus Cursor + take screenshot
- `POST /exec` - Run Windows commands
- `GET /health` - Server status

### Server Access
- Windows gateway IP: `172.29.176.1:9876`
- Start: `cd D:\code\crossauto && npm start`
