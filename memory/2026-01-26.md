## Twitter B-Roll Capture Script

Created `/root/clawd/scripts/record-broll-twitter.js` for capturing social proof footage from X:

**Features:**
- Searches X for relevant discussions using `bird-auth` (min_faves filter for engagement)
- Ranks posts by engagement score: `views + likes×10 + replies×20 + retweets×15`
- Records video of each top tweet (scrolling, hovering metrics)
- Takes screenshots of top posts
- Optional: search results compilation video

**Usage:**
```bash
node /root/clawd/scripts/record-broll-twitter.js "topic" ./output
node /root/clawd/scripts/record-broll-twitter.js "clawdbot" ./public/b-roll --max-tweets 5
```

**Output:**
- `twitter/tweet-N-username.webm` - video captures
- `twitter/tweet-N-username.png` - screenshots
- `twitter/search-topic.webm` - search results video
- `twitter/manifest.json` - metadata + rankings

**Auth:** Uses `bird-auth` wrapper at `/usr/local/bin/bird-auth` with @CubanKarel credentials

## Remotion Slide Improver Skill

Created `/root/clawd/scripts/remotion-slide-improver.js` and skill at `/root/clawdbot/skills/remotion-slide-improver/`

**What it does:**
1. After Remotion composition created by Claude Code
2. Analyzes slides from `*-transcripts.ts` file
3. For EACH slide, spawns `claude -p "Research and improve slide N..."` in Remotion dir
4. Waits for completion (up to 10 min per slide)
5. Moves to next slide
6. Tracks progress in `memory/remotion-improvements.json`

**Usage:**
```bash
node /root/clawd/scripts/remotion-slide-improver.js [composition-name]
```

**Key:** Claude Code does the actual improvement work, not this script.

Added to HEARTBEAT.md for automatic triggering after remotion sub-agent completion.
